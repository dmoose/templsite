# Welcome

This is your new site built with **templsite** - a type-safe static site generator using Go and templ.

## Getting Started

- Edit content in `content/`
- Customize components in `components/`
- Style with DaisyUI classes
- Run `make serve` to see changes live

## Commands

```bash
make build   # Build your site binary
make serve   # Start development server
make deploy  # Build static files
```
